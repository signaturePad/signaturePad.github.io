Make Change

  1. Unzip and copy the project files into the local server folder(htdocs if you are using xampp or www if you are using wamp) or push the project file into web server if you want to deploy the project into the web server. 
  2. Access the project file using appropriate URL. 
  3. Draw something on the frame and then click on the green button after that check your digital signature into 'upload' folder.

 

If you want to store signatures in a DB then there is a file name sinaturepad.db import that file into your mysql db,after that repeat the step number 1,2,3.

 

For mail functionality, I used a PHP package name PHPMailer, before trying this you need to change some parameters 

$mail->Host = 'your smtp server details';
$mail->Port = 587;
$mail->SMTPAuth = true;
$mail->Username = "Your gmail account username ";
//Password to use for SMTP authentication
$mail->Password = "Your gmail account password";
//Set who the message is to be sent from
$mail->setFrom('From emali id name','Username');
